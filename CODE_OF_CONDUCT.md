# CODE OF CONDUCT

This repository does not employ a standard corporate code of conduct. Instead, it operates on a philosophy of quiet maintenance. 

Software is inherently transient. The architecture we are parsing is a massive, decaying labyrinth built on forty years of historical accidents, and this engine, no matter how perfectly optimized, will eventually be forgotten, deprecated, or replaced. We accept this.

If you choose to interact with this repository, please observe the following:

**1. Absence of Ownership**
This project is dedicated to the public domain (CC0). Do not seek attribution, glory, or legacy here. We give the artifact away because holding onto digital things serves no lasting purpose.

**2. Quiet Contribution**
There is no need for loud debates, ideological arguments, or ego in the issue tracker. If you find a flaw in the decoding logic, simply provide the patch. Please keep your voice down. 

**3. Graceful Failure**
The engine is designed to return errors without panicking. Extend this same mechanic to human interactions. Mistakes are inevitable. Correct them without assigning moral weight, anger, or frustration. 

**4. The Work Itself**
We are simply mapping ruins so others can navigate them safely for a little while longer. Do the work because it needs to be done, ensure the parser is deterministic, and leave the code better than you found it. 

All things decay. Until then, we maintain the machine.
